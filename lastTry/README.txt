Запускать через script start в package.json

Описание JSON file:
status = Статус проекта
num = Номер проекта;
description = Описание проекта
name = Название проекта
statusId = Номер статуса

Номера статусов:
1 - Согласование
2 - В ожидании
3 - Разработка
4 - Тестирование
5 - Корректировки
6 - Готова
